# Civilizations Compared

Bilingual comparison of 70+ Western and Chinese historical figures across eight parallel eras.

[![Live Demo](https://img.shields.io/badge/demo-civilizations--compared.vercel.app-crimson)](https://civilizations-compared.vercel.app)

## Overview

A bilingual (English / Chinese) single-page site mapping Western and Chinese civilizations side by side through eight eras:

1. **Legendary Age** -- mythic founders and sages
2. **Classical Antiquity** -- philosophers and empire builders
3. **Imperial Formation** -- consolidation of power
4. **Medieval / Middle Period** -- faith, scholarship, and dynasties
5. **Renaissance / Ming** -- cultural rebirth
6. **Enlightenment / Qing** -- reason and reform
7. **Modern Era** -- revolution and modernization
8. **Contemporary** -- the global stage

Each figure includes English name, Chinese name, pinyin, and a brief description. Sticky era navigation allows quick jumping between periods.

## Design & Tech

- **Stack:** Single HTML file, vanilla JavaScript
- **Aesthetic:** Antiquarian editorial style on warm parchment
- **Color Coding:** Blue (Western) / Crimson (Chinese)
- **Typography:** Crimson Pro (English) + Noto Serif SC (Chinese)
- **Deployment:** Vercel
